# FinPay Lite

FinPay Lite is a mini FinTech web application built with ReactJS, Node.js, Sails.js and PostgreSQL.

## Tech Stack

- Frontend: ReactJS, Vite
- Backend: Node.js, Sails.js
- Database: PostgreSQL
- Authentication: JWT
- Testing: Jest, Supertest
- DevOps: Docker, GitHub Actions

## Main Features

- User registration and login
- Wallet balance management
- Money transfer
- Transaction history
- Saving goals
- Loan request workflow
- Admin dashboard
